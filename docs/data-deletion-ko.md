# 데이터 삭제 안내

GalaxyPods는 **사용자의 어떠한 데이터도 외부 서버에 저장하지 않습니다.** 모든 데이터는 사용자의 단말 안에서만 처리됩니다.

## 데이터 삭제 방법

### 방법 1. 앱 내 설정에서 삭제

1. GalaxyPods 앱 실행
2. 우측 상단 **설정** 아이콘 → **저장된 데이터 관리**
3. **마지막 위치 삭제** 버튼 → 즉시 삭제 완료

### 방법 2. 앱 제거

1. 안드로이드 홈 화면에서 GalaxyPods 아이콘 길게 누름
2. **앱 제거** 선택
3. 단말 내 모든 GalaxyPods 데이터(앱 설정, 마지막 위치 등) 자동 삭제

### 방법 3. 안드로이드 시스템 설정

1. 안드로이드 **설정** → **앱** → **GalaxyPods**
2. **저장 공간** → **데이터 삭제** 또는 **저장 공간 비우기**

---

## 외부 서버 데이터 없음

본 앱은 다음 정책을 엄격히 준수합니다.

- ✅ 사용자 식별 정보를 **수집하지 않음**
- ✅ 위치/배터리/페어링 이력을 외부로 **전송하지 않음**
- ✅ 분석/광고 SDK를 **사용하지 않음** (Crashlytics 도입 시 옵트아웃 옵션 제공)
- ✅ 사용자 계정/회원가입 **요구하지 않음**

따라서 별도의 "서버 측 데이터 삭제 요청 절차"는 존재하지 않습니다. 단말에서 데이터를 삭제하는 것이 전부입니다.
